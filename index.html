Preguntas y Respuestas
Repaso HTML, CSS y JS

¿Qué es HTML y cuál es su función principal en el desarrollo web?

HTML por sus siglas es el Lenguaje de Marcado de HiperTexto, el cual es fundamental para sentar las bases de un sitio web por su estructura y todo esto gracias a que es un lenguaje de etiquetas, el cual ayuda en la gestión apropiada de un sitio.

Pregunta 1
¿Qué son las etiquetas HTML y cómo se utilizan?

Son los elementos para darle estructura semántica a nuestro sitio web, a partir de palabras clave como: "head", "body" y "html". Estas son las que le dan la forma y estructura perfecta a cada sitio web en el internet.

Pregunta 2
Proporciona ejemplos de algunas de las etiquetas HTML más comunes.

"Body": Este representa el cuerpo del sitio web, "html": Este representa el inicio y fin del estructurado del sitio web, "h1-6": Representan los tipos de titulos de texto. "p": Simboliza un párrafo de texto el cual es una cadena de texto en tamaño pequeño, "Div": Este simboliza un Divisor el cual normalmente se utiliza para agregarle un identificador a cierta sección del sitio.

Pregunta 3
¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

CSS es el Lenguaje de Estilo de Cascada o en ingles Cascade Style Sheet, este sirve para darle un diseño único a nuestro sitio web con colores, fuentes, animaciones y todo lo que se pueda imaginar. Este se utiliza en etiquetas referenciandolas a la estructura HTML.

Pregunta 4
¿Cuáles son las propiedades más importantes de CSS?

Las propiedades mas importantes de CSS podrían ser: "backgroud-color" este le asigna un color de fondo a cierta sección referenciada por ejemplo al body por lo que el fondo de todo el cuerpo se cambiaria de color, "font-size" este cambia el tamaño de la fuente de letra, ya sea un h1 o un párrafo. "width / height" estos van de la mano ya que son los principales en los cambios de tamaño, de ancho y alto ya sea en imágenes o secciones de pantalla hasta elementos.

Pregunta 5
Explica el modelo de caja (box model) en CSS.

El modelo de caja es una forma distinta de ver un sitio web encerrando cada elemento en rectangulos el cual, muestra de que forma se comporta en el espacio de la estructura del sitio web.

Pregunta 6
¿Qué es el diseño responsivo y cómo se logra con CSS?

El diseño responsivo se le llama a la estructuracion del sitio web dependiendo del tamaño de la pantalla del dispositivo, en este caso con CSS, se utiliza media queries, el cual se le llama a los elementos y cambia ya sea el tamaño o color de texto y la seccion en la que se encuentra. Para otorgarle un diseño distinto al ver el sitio web en celular por ejemplo.

Pregunta 7
¿Qué es JavaScript y cómo se integra en el desarrollo web?

JavaScript se utiliza para crear una interacción entre el sitio web con el usuario, en este caso JavaScript es quien nos ayuda a agregarle capacidad lógica a nuestro sitio web, además de ayudar a crear componentes HTML el cual solo se manda a llamar a otras pantallas, en este caso para referenciar el codigo, se agrega la etiqueta: "script", el cual seguirá por la ruta donde se encuentra el código de lógica JS de nuestro sitio web.

Pregunta 8
Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model)./strong>

DOM interactua con el HTML Y CSS en forma de árbol, a partir del DOM podemos hacer que nuestro código JS realice cambios en el HTML o CSS cada que uno se lo indique, generando una mayor manipulación del sitio web completo desde el JavaScript. Gracias a ello la conexión entre estos 3 se vuelve bastante fuerte al emplear DOM con el creando, modificando o eliminando elementos.

Pregunta 9
Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.

Se podria crear un documento JS, con el cual el sitio web realice una acción a partir de una acción el usuario. Por ejemplo al querer calcular el total de precio de un producto descontando el 15% de descuento. JS nos ayudará en hacer una función la cual se ejecute cuando el usuario presione un botón por ejemplo y JS hará el calculo para luego mostrarlo.

Pregunta 10
¿Qué es Git y por qué es importante para el desarrollo web?

Git es una de las bases del desarrollo web, ya que será nuestro apoyo en el proceso de guardado de un proyecto en la nube. Git viene de la mano con GitHub el cual es uno de los almacenamientos más grades para los desarrolladores, ya que este proporciona un guardado seguro en versiones de proyecto. Git será la herramienta de comandos la cual nos ayudará a gestionar el guardado de nuestros proyectos.

Pregunta 11
Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web

Git maneja el control de versiones en los proyectos web, el cual consiste en guardar una versión de código más actualizada que la anterior y así sucesivamente, cada vez que un usuario guarda cambios en su código estos se almacenan encima de la versión anterior y cuando se desee volver a una versión anterior se puede hacer de una manera bastante simple. Ya sea en consola o con la interfaz gráfica.

Pregunta 12
¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

GitHub es uno de los mayores almacenamientos de proyectos web. Este fácilita la colaboración en proyectos gracias a su control de versiones y ramas de trabajo, el cual le da la extensividad necesaria para que muchas personas trabajen un mismo proyecto al mismo tiempo y corroborando la calidad a partir de las versiones. GitHub registra cada cambio que pasa dentro del repositorio y da el acceso necesario a la gente que uno desea para un trabajo eficaz.
